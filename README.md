# Dotfiles

Dotfiles を置いておきます。

自分は大きく Windows と MacOS という2つの環境を持っていて、どちらにも使える設定ファイルを目指して作っています。

どうしても環境ごとに分けたいものはファイルを分割して読み込むファイルを切り替えたりしています。
